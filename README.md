# 🎯 iLabel Sniper (iLabel 抢题助手)

**iLabel Sniper** 是一款专为 iLabel 平台设计的浏览器自动化抢题工具。它能够实时监控任务接口，并在抢到题目后自动刷新页面进入作业状态，极大提高抢题效率。

---

## ✨ 核心功能

* **自动高频抢题**：支持毫秒级任务接口监控。
* **智能状态切换**：抢到题后自动停止请求，提交作业后自动恢复抢题。
* **动态限流策略**：检测到 API 限流（429）时自动延长延迟，保护账号安全。
* **数据可视化**：内置浮动面板，实时显示请求次数、成功数及运行状态。
* **快捷键操作**：按下 `O` 键可快速显示/隐藏控制面板。

---

## 🛠️ 安装方法

1. **下载源码**：下载本项目压缩包并解压到本地。
2. **打开插件页面**：在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
3. **开启开发者模式**：在页面右上角找到 **“开发者模式”** 开关并开启。
4. **快速安装**：
* 找到解压后的 `iLabel-sniper` 文件夹。
* **直接将该文件夹拖拽到 Chrome 插件管理页面**。
* 或者点击左上角的 **“加载已解压的扩展程序”** 按钮，选择 `iLabel-sniper` 文件夹。



---

## 🚀 使用指南

1. **进入任务页**：登录 iLabel 平台，进入具体的任务/项目详情页面。
2. **启动插件**：
* 点击 Chrome 浏览器工具栏中的插件图标。
* 在弹出的窗口中点击 **“开始抢题”**。


3. **面板监控**：
* 页面右上角会出现状态面板。
* 显示 `▶️ 抢题中`：正在疯狂扫货。
* 显示 `✅ 抢到题`：页面将自动刷新，请立即开始做题。
* 显示 `📝 做题中`：此时脚本会自动保持安静，不会干扰你的作业。


4. **恢复抢题**：
* 当你提交完题目，页面出现“刷新”按钮或回到待命状态时，脚本会自动检测并继续抢题。



---

## ⚠️ 注意事项

* **保持页面活跃**：建议将该任务页独立在一个浏览器窗口中，不要最小化。
* **频率限制**：默认请求频率较高，如频繁出现 `⏸️ 限流` 状态，脚本会自动调整延迟，请耐心等待。
* **刷新弹窗**：脚本已内置屏蔽“离开此网站”弹窗的逻辑。如果刷新受阻，请检查是否有其他干扰插件。
* **单实例运行**：请勿在多个标签页同时针对同一个项目运行，以免触发平台风控。
